# CreatCV

A simple web application that allows users to enter their personal information and instantly display it in a clean, resume-style format.

link ([https://cv-app-3yzx.vercel.app/])

## Features

  - Enter basic personal details
  - Enter education and contact information
  - Live preview of resume format
  - Simple and user-friendly interface

## TechStack

  - React
  - TailwindCSS
